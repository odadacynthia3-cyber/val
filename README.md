# 💌 Valentine Web App
A simple and fun web application that allows users to generate and share a secret Valentine message with someone special ❤️.

## 🚀 Features
* 💖 Generate a unique Valentine message link
* 🔗 Share message via WhatsApp & Telegram
* 🎨 Simple and clean UI
* 📱 Mobile-friendly design

## 🛠️ Built With
* HTML
* CSS
* JavaScript
* PHP

## 📂 Project Structure
* index.php – Main page
* message.php – Displays the secret message
* style.css – Styling

## ⚙️ How It Works
1. User enters a message
2. App generates a unique link
3. User shares the link
4. Recipient opens link to view the message ❤️

## 💻 Installation
1. Clone the repository
2. Move project to your server folder (e.g. htdocs or www)
3. Start your local server (XAMPP/WAMP)
4. Open in browser
